# Wapiti

a better theme for Github Pages

[Preivew](https://aiokr.github.io)

[repo](https://github.com/aiokr/jekyll-theme-wapiti)

## 特色功能

标签云支持

单页友情链接支持

为打印机优化样式（暂时仅支持文章页）

支持PWA（仅限Chrome for Android）

# 如何使用

Fork本仓库到你的 Github 账户，并且更改仓库的名称为  ```yourname```.github.io
